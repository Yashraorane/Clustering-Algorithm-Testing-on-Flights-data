Topic: Spectral Clustering of USA Flight Data

Description:

This project aims to implement unsupervised machine learning methods on a dataset of US domestic flights from 1990 to 2009.

The dataset was sourced from the Kaggle platform, as previously indicated by the university.click here.

The primary algorithms of focus in this context will be spectral clustering. A browser-based user interface/visualizer is constructed to present the conclusions of the data.

1. Aims and Objectives

1.1. Aims

The major objective of this study is to employ unsupervised machine learning methods, particularly spectral clustering, to examine an extensive dataset of US domestic flights from 1990 to 2009.

The project intends to develop an interactive browser-based user interface that visually conveys the outcomes of the spectral clustering analysis, rendering the insights accessible and understandable to a broad audience.

1.2. Objectives

The objectives of this project are as follows:

  • Utilise unsupervised machine learning methodologies to analyse flight data, discover patterns, and discern relationships, which may include socio-economic factors, airport traffic, etc.
  • Create an interactive web interface to visualise clustering outcomes.
  • Determine the most effective clustering techniques and evaluation metrics for the specified dataset.
